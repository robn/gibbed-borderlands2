SaveEdit requires the full .NET Framework 4, which can be downloaded from
  http://www.microsoft.com/download/en/details.aspx?id=17851

Most UI interface icons are courtesy of Fugue Icons available at
  http://p.yusukekamiyamane.com/

--------------------------------------------------------------------------

Support for saves from the PlayStation Vita version:

* Loading Vita saves is supported, as is converting them to other platforms
* Saving Vita saves is not supported
* Save must already be decrypted. Your device must be running HENkaku.
* You can retrieve your save by replacing the games eboot.bin with vitashell (https://www.reddit.com/r/vitahacks/comments/50pf8q/tutorial_decrypt_save_data/)
